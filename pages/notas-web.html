    Por que testar em diferentes navegadores?

  • Só existe web se estiver dentro de um navegador;

  • Internet é uma coisa, web é outra;

  • Internet (WAN) é uma rede de computadores, de alcance mundial, engloba infraestrutura (cabos, antenas, satélites, servidores, etc...), softwares, serviços e usuários. Outros exemplos de rede: intranet, extranet. As redes de computadores se organizam por diferentes parâmetros (alcance, velocidade, quantidade de usuários, serviços oferecidos...)

  • Web é um SERVIÇO oferecido pela internet.

  • Se só acessamos a web pelos navegadores e os navegadores têm diferentes fabricantes, é FUNDAMENTAL testar os códigos realizados com HTML+CSS em diferentes navegadores.

  • Navegadores têm diferentes ENGINES (motores) de renderização.

    • Padrões são obrigatórios?

  • Não! Os padrões são BOAS PRÁTICAS que PODEM ou não ser seguidos. Certamente, o uso trará mais compatibilidade e segurança para que uma aplicação rode em qualquer navegador.

    • Por que usar indentação?

  • Indentar é uma forma VISUAL de organização do código. Através da indentação ENXERGAMOS quem pertence a quem, qual a relação de hierarquia dos elementos. Ela facilita a manutenção do código.

  • LEMBRAR: a maior parte das tags abre e fecha; poucas não fecham. Por exemplo: "<br>", "<hr>", "<input>" não fecham.

    • HTML e CSS

    (html) <meta> - A tag <meta> é usada em HTML para fornecer metadados sobre a página web. Esses metadados são informações sobre os dados da página, como o conjunto de caracteres, a descrição, palavras-chave, autor, configurações de visualização e outras informações relevantes que não são exibidas diretamente no conteúdo visível, mas são usadas por navegadores, mecanismos de busca e outros serviços da web.

    (html) <li> - A tag <li> em HTML (List Item) é usada para definir um item dentro de uma lista. Ela deve ser sempre utilizada dentro de uma lista ordenada (<ol>) ou não ordenada (<ul>) ou menu (<menu>). <strong> - A tag <strong> em HTML é usada para indicar que o texto contido possui uma importância ou relevância maior do que o texto ao seu redor. Visualmente, navegadores geralmente exibem o conteúdo da tag <strong> em negrito.

    (html) <header> - A tag <header> em HTML serve para definir o cabeçalho de uma seção ou documento, geralmente contendo elementos introdutórios ou de navegação, como logotipos, menus e títulos.

    (css) padding - é a propriedade que modifica o tamanho da caixa

    Oque é metadado?

  • Metadados, em termos simples, são dados sobre dados. São informações que descrevem um recurso digital, como um arquivo, um documento, uma imagem ou um vídeo. Estes dados adicionais, por sua vez, ajudam a identificar, organizar, encontrar e gerenciar os recursos, oferecendo informações sobre o conteúdo, como o título, o autor, a data de criação, as palavras-chave, entre outros detalhes.

    • Chaves servem para abrir e fechar BLOCOS de código.

  • Em programação, chaves ({ e }) são usadas para delimitar blocos de código. Esses blocos são conjuntos de instruções que são executadas como uma unidade, frequentemente dentro de estruturas de controle como funções, loops e condicionais.
