Sobre o Projeto DevNotes

  • DevNotes é um hub educacional colaborativo criado para facilitar o aprendizado e a troca de conhecimento entre estudantes da área de tecnologia. O projeto foi desenvolvido com foco na simplicidade, organização e interatividade.

      • O site principal funciona como uma central de anotações e comunicação com quatro seções principais:


    • Página 1: Armazena as anotações de aulas relacionadas ao Desenvolvimento Web, como HTML, CSS e JavaScript.
    • Página 2: Contém anotações voltadas à Lógica de Programação e Desenvolvimento de Software.
    • Página 3: Espaço colaborativo onde os alunos podem criar, editar e armazenar suas próprias notas.
    • Página 4: Um chat público em tempo real que permite a troca de mensagens entre os usuários, com fotos de perfil personalizadas e sistema de notificações caso a aba esteja inativa.

    • O projeto utiliza Firebase Firestore como banco de dados em tempo real, permitindo a comunicação síncrona no chat, e aproveita APIs como pravatar.cc para gerar avatares únicos para cada usuário.

    • A experiência do usuário é otimizada com foco em acessibilidade e clareza visual, aproveitando um layout responsivo e adaptado para diferentes tamanhos de tela.

      O DevNotes é uma iniciativa de código aberto, incentivando a contribuição da comunidade para aprimorar e expandir suas funcionalidades. O objetivo é criar um ambiente onde estudantes possam aprender, compartilhar e colaborar de forma eficaz.

    Para saber mais ou experimentar as funcionalidades, explore as demais páginas do site.

    • Esperamos que o DevNotes seja uma ferramenta útil e inspiradora para todos os estudantes de tecnologia!
